# 02 Appointment Scheduling

This workflow books meetings for qualified leads and communicates confirmations.

- **Inputs**: Qualified lead payload with contact details and desired meeting preferences.
- **Actions**: Create calendar events, send emails/notifications, and post status updates back to the CRM.
- **Outputs**: Calendar invite IDs, confirmation messages, and audit logs.

Use `templates/README.workflow.md` to add implementation specifics once the workflow is imported.
